# 股票块

股票块是在思源文档中直接嵌入的轻量股票挂件，用于在投资笔记、复盘日记、个股研究或个人工作台里随手记录交易、查看基本面、对比走势。

它不是行情软件，也不是数据库视图的替代品。它关注一个更窄、更顺手的场景：不用建数据库、不用离开当前文档，就能在笔记里放一个可编辑、可长期保存、多端同步的股票块。全部股票块共享同一份数据，在任意一个块里增、改、删，其他块即时更新。

> 行情、基本面数据来自东方财富公开接口，仅供个人记录与学习参考，不构成任何投资建议。

## 统一股票块

通过斜杠菜单 `/股票块`（或输入「股票」「持仓」「记录」「gupiao」「stock」）、顶栏图标菜单中的「插入股票块」插入，可在任意文档插入任意多个。每个块内置三个视图，块内切换，互不干扰：

- **记录**：一条记录 = 一只股票的一轮交易（买入 + 可选卖出）。表格支持按标的、买入日、买入价、数量、卖出日、卖出价、盈亏、盈亏%、状态（持有中 / 已清仓）排序；可就地新建、编辑、删除，并为每条记录打标签。顶部汇总「持仓 X 笔，清仓 Y 笔，总盈亏」。
- **分析**：输入代码或名称获取个股基本面，按报告期（年报 / 中报 / 一季 / 三季，多年并列）展示每股收益、净资产收益率、毛利率、负债率、营收增速、净利增速与总市值，便于横向比较与复盘。
- **对比**：把大盘指数、行业 / 概念板块、ETF、个股或某个标签下的一组股票放在同一张图里，按起点归一化到 100 进行走势对比（单选时显示绝对实值）。

## 对比视图

对比视图是本插件的重点，左侧选标的、右侧看走势：

- **五类标的**：大盘（内置上证指数、深证成指、创业板指、科创50、沪深300、中证500、中证1000 七大指数）、板块、ETF、个股、标签。板块 / ETF / 个股通过「获取」导入历史行情，可随时删除；标签页把已导入的个股按标签分组，支持整组勾选。
- **三种时间范围**，互斥切换、自动清空彼此：
  - **最近范围**：最近 1 月 / 3 月 / 6 月 / 今年 / 1 年。
  - **年份 / 季度**：鼠标悬停年份即展开该年的「全年 / 一季 / 二季 / 三季 / 四季」，自动按数据实际范围裁剪。
  - **起始日期**：选定某天到最新数据。
- **归一化对比**：多选时所有曲线从同一起点 100 出发，直观比较区间涨跌幅；单选时显示该标的的真实点位。
- **悬停十字光标**：移动鼠标即显示该日各标的的实际数值与当日涨跌幅，并按数值排序；只有当鼠标真正贴近某条折线时才单独高亮该线、淡出其余，空白处不会误高亮。
- **图例与聚焦**：底部图例可点击锁定某条折线；「清空」一键取消全部勾选，让走势图回到空白。
- 跟随挂件尺寸自适应重绘，标的多时也不串味。

## 核心功能

- 一份数据，多个块、多个视图实时同步，不依赖思源数据库。
- 行情与基本面数据通过思源内核的网络代理向东方财富拉取，规避浏览器跨域限制；K 线在多个备用域名间自动重试。
- 内置七大宽基指数行情，开箱即用；个股 / 板块 / ETF 由用户按需导入，导入区间默认最近一年，可在「获取」时自定义月数 / 年数。
- 独立「股票」面板：可从顶栏图标菜单或命令面板打开放大版界面，在思源标签页中以更舒展的宽度展示同一份数据，适合集中查看与对比。
- 跟随思源明暗主题：嵌入块与独立面板都会实时跟随思源的浅色 / 暗色主题（含系统昼夜自动切换）切换配色。
- 记录、个股均可打标签；两级标签管理支持新增、重命名、删除，删除标签后会从所有记录上摘除。
- 数据存放在独立的工作空间文件中，随思源官方云同步多端同步；写入前自动备份，文件损坏时禁止写入以防覆盖。

## 数据存储

股票数据统一保存到思源工作空间：

`/data/storage/stock-block/data.json`

每次写入前会把上一份正确数据备份到同目录的 `data.json.bak`；当文件损坏或读取异常时自动禁止写入，避免覆盖原有数据。多个块同时打开时通过 BroadcastChannel 即时同步，跨设备依赖思源官方云同步。

## 适用场景

- 投资笔记与复盘日记里的持仓记录与盈亏统计。
- 个股研究文档中的基本面对比与走势对比。
- 把某个赛道（如「机器人」「AI 算力」）下的几只股票放在一张图里横向比较。
- 个人工作台中的市场概览。
- 任何希望把行情与持仓直接放在笔记内容里、又不想另开行情软件的场景。
