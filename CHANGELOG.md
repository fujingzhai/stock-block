# 更新日志

## 0.3.4

- **支持分批卖出**：一条记录可录入多笔卖出（日期/卖价/数量各一行），新建/编辑表单新增「添加卖出」，自动校验累计卖出数量不超过买入量；旧版单次卖出数据（sellDate/sellPrice/sellQty）自动迁移为分批结构并向后兼容
- **状态按剩余持仓判定**：卖出数量未覆盖全部买入时显示「部分卖出」，全部卖完显示「清仓」，未卖显示「持有」；已实现盈亏按各笔卖价分别计算后汇总
- **记录表格列重排**：拆分为「卖出 / 卖价 / 卖出量」三列分别展示（多笔分行显示），并独立出「盈亏% / 状态 / 现价」列；表头与列宽按宽面板、窄挂件两套尺寸自适应
- **现价列收窄**：宽面板下现价列由大比例收到仅够显示价格与日期的小宽度，窄挂件下保留较宽以免截断
- **记录表统一左对齐**：表头与全部内容列统一左对齐（盈亏红绿色保留），仅最右操作列贴右；修复此前卖出/卖价/卖出量三列内容仍右对齐、与占位「—」左右混杂的显示错乱
- **操作列交互修复**：编辑/删除按钮仅在 hover 行时显示，非 hover 不占位、不再触发省略号；调整操作列宽度，保证窄挂件下删除按钮完整可见
- **挂件高度自适应兜底**：新增内容尺寸监听（ResizeObserver），记录/分析/对比任一视图内容长高（新增行、异步加载的图表/字体等）时都会自动重新撑开挂件，避免出现内部滚动槽

## 0.3.3

- **持仓表格列重新布局**：买入日→买入时间，买入价→成本，卖出日与状态合并为「状态」列（清仓显示日期+清仓，持有显示持有中），卖出价→卖价/现价
- **持仓盈亏自动跟踪现价**：持有中的记录自动从已导入的 K 线数据读取最新收盘价计算浮动盈亏，清仓记录仍按输入的卖价计算已实现盈亏；顶部汇总包含浮动盈亏
- **新建记录表单优化**：股票输入框提示改为「名称」（去掉误导性的代码提示），买入/卖出标题改用中性色（红绿仅用于盈亏显示）

## 0.3.2

- **修复刷新覆盖历史数据**：刷新时不再用固定 1 年范围拉取后整体覆盖，改为根据每只股票已有 K 线的最早日期自动推算范围，新旧数据按日期合并去重，历史走势不再丢失
- **修复并发重复条目**：数据加载时自动对 marketSeries 和 individualStocks 按 (kind, code) 去重（保留 K 线最多的一条），upsert 方法增加 push 前二次检查，防止快速连续操作产生重复记录
- **新增时间范围选项**：个股与对比视图均增加「最近 2 年」「最近 3 年」，完整顺序为：今年以来、1 月、3 月、6 月、1 年、2 年、3 年、全部范围
- **默认时间范围调整**：个股与对比视图的默认显示范围从「最近 1 年」改为「今年以来」
- **「全部范围」改为显示全部已有数据**：不再硬编码从 2020 年开始，而是展示该标的从最早到最新的全部 K 线
- **获取对话框类型改为必选**：去掉空白的「选择类型」占位项，默认选中「大盘」；始终使用手动指定的类型解析标的，不再走自动识别逻辑，避免科创 100 等非主流指数被误归为板块
- **移除 migrateLegacy 强制分类逻辑**：不再在每次加载数据时将非内置指数强制转为「板块」，确保用户手动指定的「大盘」类型（如科创 100）在刷新后保持不变
- **对比视图增加「全部范围」**：RECENT_RANGES 末尾补齐「全部范围」选项，与个股视图保持一致
- **对比视图列表支持排序**：大盘/板块/ETF/个股列表顶部新增排序下拉框，支持按音序或代码排序，各面板独立记忆排序偏好
- **「最近范围」改为自定义下拉菜单**：替换原生 `<select>` 为按钮触发 + 绝对定位菜单面板，始终向下弹出，解决面板模式下选项列表方向不可控的问题
- **修复下拉菜单交互体验**：触发器与菜单之间补隐形桥接消除 hover 间隙，选完选项或点击外部自动收起菜单

## 0.2.0

- 对比视图「起始日期」改为原地更新，不再每输入一位数字就因控件重建而丢失焦点，可正常连续输入年月日
- 修复「最近范围」与「起始日期 / 年份季度」的状态冲突：非最近模式下下拉显示占位项，切回任一最近范围都会真实触发并清空已选的具体日期
- 修复「年份 / 季度」子菜单展开困难：跨年份按钮与季度子菜单之间补隐形悬停桥，且选择后不再重建 DOM，鼠标悬停即展开、无需先点一下

- 走势图悬停聚焦改为带阈值：只有鼠标真正贴近某条折线时才单独高亮该线、淡出其余，空白处不再误高亮
- 新增「清空」按钮：一键取消全部勾选标的，让走势图回到空白；并与底部图例统一样式
- 优化对比视图左侧「大盘 / 板块 / ETF / 个股 / 标签」五个标签为等宽三列网格，与上方「记录 / 分析 / 对比」对齐更整齐
- 补全上架元素：中英文 README、独立图标与预览图、LICENSE、CHANGELOG、发布工作流
